# Domain Controllers

A flat list of every known domain controller in the forest and the site it currently belongs to — the same
data source feeding the FSMO target dropdowns, the Replication diagram, and the
[Server Switcher](docs/dashboard/server-switcher.md).

---

## Moving a DC Between Sites

A domain controller's site placement can be changed here — useful after a subnet's site mapping changes, or
when correcting a DC that provisioned into the wrong site.
