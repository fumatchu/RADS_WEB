# Logs

A live log viewer for RADS-WEB and the services it manages — useful for troubleshooting without needing shell
access to the box.

---

## What You Can View

Logs are grouped by service, so you can jump straight to Samba, Apache, DHCP, DNS, or RADS-WEB's own
application log without hunting through `/var/log` by hand.
