# Generate RSAT File

Produces a connection file for Microsoft's Remote Server Administration Tools (RSAT), for admins who want to
manage this domain from a Windows workstation alongside RADS-WEB rather than instead of it.

---

## Using It

Generate the file from this page and import it into RSAT on a Windows machine. This is a convenience — it
doesn't change anything about how the domain itself is administered, and RADS-WEB remains fully capable of
every day-to-day AD task on its own.
