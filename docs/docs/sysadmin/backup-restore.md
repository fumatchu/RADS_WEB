# Backup / Restore

Back up and restore the Samba AD database and RADS-WEB's own configuration from one place.

---

## What's Covered

- The Samba Active Directory database (`sam.ldb` and related files)
- RADS-WEB's own application configuration

---

## Recommended Practice

Take a backup before any major change — an FSMO seize, a schema modification, or a manual edit outside the
normal UI flows. A recent backup is the fastest way back to a known-good state if something goes wrong.
