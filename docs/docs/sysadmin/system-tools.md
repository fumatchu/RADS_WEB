# System Tools

A grouped landing page for day-to-day server administration outside of AD content itself.

---

## What's Grouped Here

| Section | Covers |
|---|---|
| Server Monitor | Live resource usage — CPU, memory, disk, and related system stats |
| Server Administration | General server-level administrative actions |
| Active Directory Management | Quick links into DNS and DHCP without leaving System Tools |
| Network Services | Status and control for Samba, DNS, NTP, DHCP, Firewall, and Fail2Ban |

---

## Network Services

Each service shown here reflects the same underlying state as the
[Directory Health](docs/dashboard/directory-health.md) and Network Services tiles on the Dashboard — this is
the place to actually start, stop, or reconfigure a service, rather than just observe its status.
