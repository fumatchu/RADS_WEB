# DHCP Overview

RADS-WEB manages **Kea DHCP4** — ISC's modern DHCP server — for IPv4 scopes, options, and lease tracking.

---

## Where to Find It

DHCP lives under System Tools → Network Services, alongside DNS, Firewall, and the other network-facing
services this server runs. The sidebar's **DHCP** shortcut jumps straight there.

---

## What You Can Do

- Configure DHCP scopes and their options
- View active leases in real time
- Filter leases by subnet
- Jump directly to a scope's paired DNS zone (and back) when dynamic DNS is in use

See [Scopes & Leases](docs/dhcp/scopes-leases.md) for the day-to-day screen, and
[DNS / DHCP Integration](docs/dhcp/dns-integration.md) for how scopes and zones stay linked.
