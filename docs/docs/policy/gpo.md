# Group Policy Objects

Create, link, and edit Group Policy Objects (GPOs) directly from the browser.

---

## Managing GPOs

- **+ New GPO** — create a new, unlinked GPO
- **↻ Refresh** — reload the GPO list from AD
- Each GPO can be opened in an editor for its settings, and managed via a **Links** view showing every OU,
  domain, or site it's currently linked to

---

## GPO Links

The Links modal lets you add or remove links between a GPO and the containers it should apply to, without
needing a separate tool.
