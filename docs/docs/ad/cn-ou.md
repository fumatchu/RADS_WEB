# CN / OU

Browse and manage the domain's container (CN) and organizational-unit (OU) structure.

---

## The List View

- **+ Create** — create a new container or organizational unit
- **↻ Refresh** — reload from AD
- Filter box — search by name
- **CN** / **OU** pills — toggle which object type is shown
- **⚙ Advanced** — reveals AD's built-in system containers, hidden by default to keep the list focused on the
  containers you actually manage day to day

---

## Directory Tree

Below the list, a collapsible **Directory Tree** renders the full CN/OU hierarchy so you can see nesting and
where a given container sits relative to the rest of the domain at a glance.
