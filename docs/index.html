ROCKY ACTIVE DIRECTORY SERVER
RADS-WEB
Documentation

RADS-WEB is a self-hosted dashboard for managing a Samba Active Directory domain controller — users & groups, DNS, DHCP, Group Policy, Sites & Services, replication health, and day-to-day server administration, all from a browser. Use the sidebar to browse install instructions and a full walkthrough of every feature, or press Ctrl+K to search.
